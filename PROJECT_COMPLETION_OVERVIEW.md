## 🚀 快速啟動指南

### 3 步啟動系統

```bash
# 第 1 步: 啟動後端 (提供 API + 靜態文件)
npm run server

# 第 2 步: 新建終端，啟動前端開發服務
npm run dev

# 第 3 步: 打開瀏覽器
http://localhost:3001/admin-system.html
```
