# 管理後台 UI 更新 - Layui Admin 風格

## 概述
已成功將管理後台 `AdminDashboard.tsx` 從深色漸變主題重新設計為 Layui Admin 風格的清爽淺色主題。

## 更新內容

### 環境資訊
- **前端框架**: React 19.0, React Router 7.13, TypeScript 5.8, Vite 6.2, Tailwind CSS 4.1.14
- **後端框架**: Express.js 4.21.2, SQLite
- **開發端口**: Frontend 3000, Backend 3001

### UI 主要變更

#### 1. 顏色管理 (Design System)
- **前背景**: 純白色背景 (bg-white, bg-gray-100)
- **文本色**: 深灰色 (text-gray-700, text-gray-800)
- **邊框色**: 淺灰色邊框 (border-gray-200, border-gray-300)
- **強調色**: 藍色 (blue-500) 為主、綠色 (green-100) 為成功、紅色 (red-50) 為危險

#### 2. 佈局架構
```
頂部導航欄 (80px)
├── 側邊欄 (256px 寬)
│   └── 導航菜單
│       ├── 📊 概覽
│       ├── 👥 會員管理  
│       ├── 📦 訂單管理
│       └── 設置選項
└── 主要內容區域
    ├── 概覽頁面 (統計卡片 + 系統狀態)
    ├── 會員管理 (搜尋 + 表格)
    └── 訂單管理 (狀態管理表格)
```

#### 3. 核心組件

**統計卡片** (Overview Tab)
- 會員總數、訂單總數、包裹總數
- 圓形圖標背景
- 簡單文本和增長百分比

**用戶表格** (Users Tab)
- 搜尋欄: 用戶名/手機/郵箱
- 表格列: ID, 用戶名, 手機, 郵箱, 姓名, 註冊日期, 操作
- 輕量級悬停效果
- 刪除操作带確認對話框

**訂單表格** (Orders Tab)
- 訂單ID, 用戶ID, 金額, 狀態, 建立日期, 詳情按鈕
- 狀態下拉選單 (待處理/處理中/已發貨/已交付/已取消)
- 狀態颜色编码:
  - pending: 黃色背景
  - processing: 藍色背景
  - shipped: 紫色背景
  - delivered: 綠色背景
  - cancelled: 紅色背景

#### 4. 響應式設計
- 統計卡片: 3列桌面 → 1列移動設備
- 表格: 按需滾動
- 側邊欄: 可折疊設計 (待實現)
